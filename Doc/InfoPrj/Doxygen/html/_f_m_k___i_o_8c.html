◆ FMKIO_Set_OutPwmSigCfg()

t_eReturnState FMKIO_Set_OutPwmSigCfg ( t_eFMKIO_OutPwmSig f_signal_e,
t_eFMKIO_PullMode f_pull_e,
t_uint32 f_frequency_u32,
t_uint16 f_dutyCycle_u16,
t_bool f_startNow_b )

Set an intput in PWM configuration.
.

Note
Allow the user to choose among pwm intputs.
This function configure bspInit , call HAL_function, call FMKCPU to configure a timer in order to convert generate the pwm period and dutycycle.
IMPORTANT NOTE -> For SMT32, each signal has a timer and a channel based on confifguration. A timer has multiple channel that shared the frequency f_frequency_u32, once the timer set with this frequency the other signal (channel) will have the same frequency.
If the frequency is changed, it will be changed for every signal.
Parameters
[in]f_signal_e: the input analog signal, a value from t_eFMKIO_OutPwmSig
[in]f_pull_e: the input pull mode, value from t_eFMKIO_PullMode
[in]f_frequency_u32: the PWM frequency
[in]f_dutyCycle_u16: the dutycyle, value between 0 (0%) - 1000 (100%)
[in]f_puf_startNow_bll_e: wether or not the pulse generation should start immediately
Return values
RC_OKRC_OK
RC_ERROR_PARAM_INVALIDRC_ERROR_PARAM_INVALID
