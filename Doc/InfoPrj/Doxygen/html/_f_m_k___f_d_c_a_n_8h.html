t_sFMKFDCAN_RxItemEventCfg f_RxItemCfg_s )

Configure a RxItem to be received.
.

Note
This Function is used for registration to received a message CAN.
If a Can message is received on the node f_Node_e, and the identifier matches the identifer & mask from registration, software will call you with t_cbFMKFDCAN_RcvItem.
The registration has a limitation, if the Registration buffer is pass,* RC_ERROR_LIMIT_REACHED is return.
