◆ FMKCDA_Cyclic()

t_eReturnCode FMKCDA_Cyclic ( void )

Perform all cyclic action to this module

Note
Every Cycle in OPE_MODE, this function start Adc conversion if the adc is config as so (Interruption or Dma). If a previous conversion is finished, this function store the value in the right channel and update flag in consequence.
This function also perform cyclic diagnostic on channel every x seconds, parameter reference in configPrivate.
Return values

Function to get the analog value from a adc channel.

Note
This function has to be used in every adc configuration except the ones involving interrupt.
if the conversion of adc is done and the value is updated, this function store the rawAnalog value in f_AnaMeasure_u16.

◆ FMKCDA_Set_AdcChannelCfg()

t_eReturnCode FMKCDA_Set_AdcChannelCfg ( t_eFMKCDA_Adc f_Adc_e,
t_eFMKCDA_AdcChannel f_channel_e,
t_eFMKCDA_HwAdcCfg f_adcCfg_e )

Function to add a adc channel configuration to an adc instance.

Note
If the Adc config is not congigured yet it will be with f_adcCfg_e.
Keep in mind that once the ADC is configured with a config every channel of the ADC has the same configuration.
That means every channel from an ADC should have the same hardware config.
If not, a error is reach.
Parameters
[in]f_Adc_e: enum adc, value from t_eFMKCDA_Adc
[in]f_channel_e: enum adc channel, value from t_eFMKCDA_AdcChannel
[in]f_adcCfg_e: enum adc config, value from t_eFMKCDA_HwAdcCfg
Return values
RC_OKRC_OK
RC_WARNING_WRONG_STATERC_ERROR_WARNING_STATE
RC_ERROR_ALREADY_CONFIGUREDRC_ERROR_ALREADY_CONFIGURED
