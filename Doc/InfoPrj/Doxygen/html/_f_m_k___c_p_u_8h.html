◆ FMKCP_Set_EvntTimerCfg()

t_eReturnCode FMKCP_Set_EvntTimerCfg ( t_eFMKCPU_InterruptLineEvnt f_EvntITLine_e,
t_uint32 f_periodms_u32,
t_cbFMKCPU_InterruptLine f_ITChannel_cb )

Configure a timer channel on event configuration.
.

Note
This function initialize the timer in event configuration if the timer is not configured yet.
Once the timer configure is done, update the channel state using function "FMKCPU_Set_InterruptLineOpe" and every f_periodms_u32 callback function is called.
IMPORTANT, In hardware this is the Timer which manage the interruption with ARR register, which means, it is best that the timer is a basic timer with only one channel, 'cause the others will be unused.
Parameters
[in]f_EvntITLine_e: enum value for Interrupt Line, value from t_eFMKCPU_InterruptLineEvnt
[in]f_periodms_u32: period before calling function, in millisecond
[in]f_ITChannel_cb: callback function to call

◆ FMKCPU_Set_ICChannelCfg()

t_eReturnCode FMKCPU_Set_ICChannelCfg ( t_eFMKCPU_InterruptLineIO f_InterruptLine_e,
t_eFMKCPU_ChnlMeasTrigger f_MeasTrigger_e,
t_cbFMKCPU_InterruptLine f_ITChannel_cb )

Configure a timer channel in Input Compare configuration.
.

Note
The IC Timer configuration is used to measure a signal frequency.
This function initialize the timer in Input Compare confgiuration if the timer is not configured yet.
IMPORTANT, the IC config is based on a timer configuration which share multiple channels, ina sense that, frequency is shared by all PWM channels.
In result, the modification of the timer configuration reverbate for all channels.

Once the timer is init correctly, this function set f_channel_e configuration based on the interrupt mode gives by f_MeasTrigger_e.
Finally, if the CPU detect the event, it will call the function f_ITChannel_cb.
Parameters
[in]f_InterruptLine_e: enum value for Interrupt Line, value from t_eFMKCPU_InterruptLineIO
[in]f_MeasTrigger_e: trigger for interruption, value from t_eFMKCPU_ChnlMeasTrigger
[in]f_ITChannel_cb: call back function to call
Return values

RC_ERROR_PTR_NULLRC_ERROR_PTR_NULL

◆ FMKCPU_Set_PWMChannelCfg()

t_eReturnCode FMKCPU_Set_PWMChannelCfg ( t_eFMKCPU_InterruptLineIO f_InterruptLine_e,
t_uint32 f_pwmFreq_u32 )

Configure an tinterrupt line in PWM configuration.
.

Note
First, this configuration set the bsp timer cfg in PWM mode.
Once it's done, this function configure the bsp channel in PWM mode too.
IMPORTANT, the PWM generation is based on a timer configuration which share multiple channels, in a sense that, frequency is shared by all PWM channels.
In result, the modification of the timer configuration reverbate for all channels.

For Instance, every FMKCPU_INTERRUPT_LINE_IO_0Y, Y belong to [0, Innfini] shared the same timer.
