Set a timer channel in Input Compare configuration.
.

Note
The IC Timer configuration is used to measure a signal frequency.
This function initialize the timer in Input Compare confgiuration if the timer is not configured yet.
Once the timer is init correctly, this function set f_channel_e configuration based on the interrupt mode gives by f_MeasTrigger_e.
Finally, if the CPU detect the event, it will call the function f_ITChannel_cb.

t_eFMKCPU_InterruptChnl f_channel_e,
t_uint32 f_pwmFreq_u32 )

Configure a timer channel in PWM configuration.
.

Note
First, this configuration set the bsp timer cfg in PWM mode.
Once it's done, this function configure the bsp channel in PWM mode too.
IMPORTANT, the PWM generation is based on a timer configuration which share multiple channel, as so, frequency is shared by all PWM channel.
In result, the modification of the timer configuration reverbate for all channels.

[in]f_channel_e: enum value for the channel, value from t_eFMKCPU_InterruptChnl

◆ FMKCPU_Set_PWMChannelDuty()

t_eReturnState FMKCPU_Set_PWMChannelDuty ( t_eFMKCPU_Timer f_timer_e,
t_eFMKCPU_InterruptChnl f_channel_e,
t_uint16 f_dutyCycle_u16 )

Set the DutyCycle to a timer channel.

Note
Modify the CCRx register to change the dutyCycle.
If the timer channel is disable, and the value is diferrent from 0, this function will enable the channel and set dutycycle give by f_dutyCycle_u16.
If the value equals 0 this function will stop the PWM pulse (channel).
